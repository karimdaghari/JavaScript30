* `Array.splice()`
```javascript
    Array.splice(index, n);
    /* index: if it's negative, then index is `Array.length`
    Example: Array.splice(-1).
    Suppose that Array.length === 3.
    `Array.splice`'s index will be from Array[1]. */

    /* n: number of elements to delete, 
    Starting from index + 1.
    */
```
